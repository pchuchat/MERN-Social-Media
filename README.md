# MERN Social Media App

## Yleiskuvaus

Tämä on MERN-pohjainen sosiaalisen median sovellus, joka sisältää autentikaation, tykkäystoiminnon sekä tumman tilan tuen. Käyttöliittymä on rakennettu Reactilla ja Material-UI:lla.
